# mvsolver - 「Minesweeper Variants」を自動で解くプログラム

## 概要
Steamで公開されているゲーム「Minesweeper Variants」を、プログラム実行後、画像の判別と画面のクリックを自動で行いゲームを解きます。
動作に甘い点が多く、自分の環境でしか試していないので下記の注意点をお読みください。プルリクエストも大歓迎です。
現在解けるのはルール「V:バニラ」のみです。

## 使い方
「Minesweeper Variants」を起動し、解きたいレベルを選択した後、盤面が表示されている状態でこのプログラムを実行します。
同レベル内なら、解き終わった後自動で次のステージに移動し解き続けます。

## 注意点
1. マウス操作を自動で行うので、初めて実行する際はループ回数を少なめにし、動作するかどうかを確認してみてください。
2. 「Minesweeper Variants」の画面はすべて画面に映してください。一部が隠れていたりすると動作しません。
3. 5x5はすぐ解けますが、8x8だと1盤面に15～20分ぐらいかかります。
